Bindet via Shortcode ein Rafflecopter-Widget in Artkeln und Seiten ein.


== Description ==

"WP-Rafflecopter" ermöglicht es, via Shortcode ein [Rafflecopter](http://www.rafflecopter.com "Rafflecopter")-Widget in Artkeln und Seiten einzubinden.

= Einbindung =

Die Einbindung erfolgt via Shortcode mittels Button im WYSIWYG-Editor, dabei wird lediglich die ID des Giveaway benötigt. Diese findet ihr in euren Rafflecopter-Giveaway-Optionen im HTML-Code:

Beispiel: &lt;a id="rc-xy0a123" href="http://www.rafflecopter.com/rafl/display/xy0a123/" rel="nofollow"&gt;a Rafflecopter giveaway&lt;/a&gt;

In diesem Falle wäre die ID xy0a123, folglich lautet der Shortcode wie folgt:

[rafflecopter xy0a123]

3. Das Plugin in WordPress aktivieren
